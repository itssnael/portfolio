Sports app - Integration

About the project

The goal was to design and integrate the interface for a collaborative sports app. The concept? Connecting users with similar fitness goals, allowing them to share their performances, workout routines, and nutrition tips.

The Process

Before diving in, I conducted visual research on Behance and Pinterest to establish a modern and dynamic visual direction tailored to the fitness industry. Based on the brief, I designed the screens on Figma. I focused on the user experience to highlight the core features, the workout list system, user profiles, and the chat area. Once the mockup was finalized, I built a clean, semantic HTML structure and styled it using CSS.

The Pumpkin Killer

About the project

Developed in Unity, this survival 3D horror game focuses on collision management and UI interactions. The player explores a castle and must collect three flames, which appear successively by triggering sounds and visual effects. The game consists of three stages: a start screen with a launch button, the gameplay phase in FPS, and an end screen.

The Process

I chose to create a Survival Horror game with an Halloween theme. The story follows a father trying to find his daughter, kidnapped by the Pumpkin Killer on Halloween night.

Exotic flowers - Ajax

About the project

Built as a dynamic one-page website, this platform utilizes Ajax to enable smooth image filtering based on several criteria. The goal was to create an ergonomic interface highlighting exotic flora.

The Process

I developed a one-page website about exotic flowers, integrating Ajax filtering by habitat and usage. The site uses HTML, CSS, JavaScript (jQuery, Ajax), PHP, and a database I created to dynamically display and sort data. I also integrated an interactive SVG map that allows users to filter flowers by their geographical area.
